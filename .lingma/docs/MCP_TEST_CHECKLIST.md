### 测试 3: Shell MCP（可选）

**前提**: 已启用 Shell MCP

**启用步骤**:

编辑 `.lingma/config/mcp-servers.json`:
```json
{
  "mcpServers": {
    "shell": {
      "disabled": false  // ← 改为 false
    }
  }
}
```

重启 IDE。

**测试步骤**:

1. 在 IDE 中打开智能会话
2. 切换到智能体模式
3. 输入以下提示词：
