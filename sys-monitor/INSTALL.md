# SysMonitor 系统监控工具

## 安装说明

### 方法 1：MSI 安装包（推荐）

1. 找到安装包：
   - 文件位置：`sys-monitor\src-tauri\target\release\bundle\msi\SysMonitor_0.1.0_x64_en-US.msi`
   - 文件大小：4.8 MB

2. 双击运行 MSI 安装包

3. 按照安装向导完成安装

4. 安装完成后，可以在开始菜单或桌面找到 SysMonitor 快捷方式

### 方法 2：独立可执行文件

1. 找到可执行文件：
   - 文件位置：`sys-monitor\src-tauri\target\release\sys-monitor.exe`
   - 文件大小：12.6 MB

2. 双击运行即可启动，无需安装

## 功能特性

### 实时监控
- **CPU 使用率**：每秒更新，显示全局使用率和历史趋势
- **内存使用率**：实时监控系统内存使用情况
- **磁盘使用**：显示所有磁盘的容量、使用量和剩余空间

### 可视化图表
- CPU 使用率历史曲线（1 分钟）
- 内存使用率历史曲线（1 分钟）
- 磁盘使用量柱状图（堆叠显示已用/空闲）
- 每个磁盘的详细信息卡片

### 技术特性
- 1 秒轮询间隔，实时数据更新
- SQLite 数据库存储历史数据
- 支持深色模式
- 响应式设计，适配不同窗口大小

## 界面说明

### CPU 监控卡片
- 显示当前 CPU 使用率百分比
- 曲线图显示最近 1 分钟的使用趋势

### 内存监控卡片
- 显示当前内存使用情况
- 曲线图显示内存使用百分比趋势

### 磁盘使用卡片
- 柱状图对比各磁盘使用率
- 详细卡片显示：
  - 磁盘挂载点（如 C:\）
  - 磁盘类型（SSD/HDD）
  - 已用空间
  - 空闲空间
  - 总容量
  - 使用百分比进度条

## 系统要求

- **操作系统**：Windows 10/11 (64 位)
- **内存**：至少 512 MB 可用内存
- **磁盘空间**：至少 50 MB 可用空间

## 技术信息

- **版本**：0.1.0
- **构建时间**：2026-04-13
- **架构**：x64
- **框架**：Tauri v2.10.3 + React + Rust

## 注意事项

1. 首次启动可能需要几秒钟初始化数据库
2. 应用会在后台持续收集系统指标
3. 历史数据存储在 SQLite 数据库中
4. 深色模式跟随系统设置

## 故障排除

### 应用无法启动
- 确保已安装 Windows 10/11 64 位系统
- 检查是否有杀毒软件阻止
- 尝试以管理员身份运行

### 数据显示不正确
- 关闭应用后重新启动
- 检查是否有其他监控软件冲突

### 性能问题
- 应用占用资源极少，通常不会影响系统性能
- 如遇卡顿，可尝试重启应用
